Solving Jigsaw-Like Tcaptcha

class qqqr.up.captcha.jigsaw.Piece(img, piece_pos)

Represents the jigsaw piece.

crop()

The crop method crops the image and saves the contour of the jigsaw piece. It also saves a bounding box for that contour, and calculates padding to be used in cropping.

Use .bbox to get bounding box of the piece, and .padding to get padding size.

img

BGR channel masked by mask

imitated(a=0.3)

This method attempts to generate a piece view like that on the puzzle. In order to help cv2.matchTemplate to get an accurate result.

It will dim the original piece and draw its contour with white lines.

Parameters

a (float) – coeff to be multiplied with the image in order to dim it, default as 0.3

Returns

generated piece image.

mask

alpha channel

strip()

The strip method crops the image and returns exactly the piece w/o any padding.

Once cropped, use .bbox to get bounding box of the piece, and use .padding to get padding size.

strip_mask()

Generate a mask for cv2.matchTemplate since the piece is in an irregular shape.

qqqr.up.captcha.jigsaw.imitate_drag(x)

The imitate_drag function simulates a drag event.

The function takes one argument, x, which is the number of pixels that the user drags. The function returns a list of lists containing three integers: [x_coordinate, y_coordinate, time]. Each coordinate and time value is randomly generated according to corresponding rules.

Parameters

x (int) – Specify the number of pixels that the user drags.

Return type

List[List[int]]

Returns

A list of lists, where each sublist contains three elements: the x coordinate, y coordinate and time
