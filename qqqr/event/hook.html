事件与回调

aioqzone has designed a simple hook management system. You may extend your own event, or just inherit a event and implement your hooks.

Define hooks that can trigger user actions.

class qqqr.event.Emittable

An object has some event to trigger.

clear(*hook_cls, cancel=True)

Clear the given task sets

参数

  • hook_cls (str) – task class names

  • cancel (bool) – Cancel the task if a set is not empty, defaults to True. Else will just clear the ref.

async wait(*hook_cls, timeout=None)

Wait for all task in the specific task set(s).

参数

timeout (float, optional) – timeout, defaults to None

返回类型

Tuple[Set[Task], Set[Task]]

返回

done, pending

备注

If timeout is None, this method will loop until all tasks in the set(s) are done. That means if other tasks are added during awaiting, the added task will be waited as well.

参见

asyncio.wait()

class qqqr.event.Event

Base class for event system.

class qqqr.event.EventManager

EventManager is a convenient way to create/trace/manage friend classes.

sub_of(event)

Use this method to get current inheritence from an ancestor which was given to the factory.

>>> class Mgr(EventManager[Event1, Event2]): pass
>>> m = Mgr()
>>> m.sub_of(Event1)    # must be one of Event1, Event2
Event1
返回类型

Type[TypeVar(Evt, bound= Event)]
