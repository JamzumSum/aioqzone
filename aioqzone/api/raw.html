  • typeid (int) – typeid

  • topicId (str) – topic id, got from html

  • uin (Optional[int]) – host uin, defaults to None, means current logined user.

This api is a ‘slow’ api, means it always returns an error code, requesting a retry a moment later. Use emotion_msgdetail() to get raw photo/video in a feed.
