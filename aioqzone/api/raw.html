Qzone API

class aioqzone.api.raw.QzoneApi(client, loginman)

Just a wrapper for Qzone http interface. No type validating.

_relogin_retry(func)

A decorator which will relogin and retry given func if cookie expired.

‘cookie expired’ is indicated by:

  • aioqzone.exception.QzoneError code -3000 or -4002

  • HTTP response code 403

参数

func (Callable) – a callable, which should be rerun after login expired and relogin.

备注

Decorate code as less as possible

警告

You SHOULD NOT wrap a function with mutable input. If you change the mutable var in the first attempt, in the second attempt the var saves the changed value.

_rtext_handler(rtext, cb=True, errno_key=('code', 'err'), msg_key=('msg', 'message'))

Handles the response text recieved from Qzone API, returns the parsed json dict.

参数

  • rtext (str) – response text

  • cb (bool) – The text is to be parsed by callback_regex, defaults to True.

  • errno_key (Tuple[str, …]) – Error # key, defaults to (‘code’, ‘err’).

  • msg_key (Tuple[str, …]) – Error message key, defaults to (‘msg’, ‘message’).

抛出

aioqzone.exception.QzoneError – if errno != 0

返回类型

Dict[str, Union[bool, int, str, Dict[Union[str, int], Union[bool, int, str, Dict[Union[str, int], ForwardRef], List[ForwardRef]]], List[Union[bool, int, str, Dict[Union[str, int], ForwardRef], List[ForwardRef]]]]]

返回

json response
