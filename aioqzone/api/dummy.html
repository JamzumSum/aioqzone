async floatview_photo_list(album, num)

Get detail of an album, including raw image url.

Parameters

  • album (AlbumData) – Necessary album data

  • num (int) – pic num

Raises

  • httpx.HTTPStatusError – error http response code

  • aioqzone.exception.QzoneError – error qzone response code

  • qqqr.exception.UserBreak – qr login canceled

  • aioqzone.exception.LoginError – not logined

  • SystemExit – unexcpected error

  • aioqzone.exception.CorruptError – maybe data is corruptted

Return type

List[FloatViewPhoto]

Returns

album details

Warning

This api is a ‘slow’ api, means it always returns an error code, requesting a retry a moment later. Use emotion_msgdetail() to get raw photo/video in a feed.

Return type

FeedsCount

Returns

Note

This api is also the ‘keep-alive’ signal to avoid cookie from expiring. Call this api every 300s can help keep cookie alive.
