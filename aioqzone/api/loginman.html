Login Managers

Collect some built-in login manager w/o caching. Users can inherit these managers and implement their own caching logic.

class aioqzone.api.loginman.MixedLoginMan(client, uin, strategy, pwd=None, refresh_time=6)

A login manager that will try methods according to the given QrStrategy.

class aioqzone.api.loginman.QRLoginMan(client, uin, refresh_time=6)

Login manager for QR login. This manager may trigger LoginSuccess() and LoginFailed() hook.

async _new_cookie()
Raises

  • qqqr.exception.UserBreak – qr polling task is canceled

  • _NextMethodInterrupt – on exceptions do not break the system, such as timeout, Http errors, etc.

  • SystemExit – on unexpected error raised when polling

Return type

Dict[str, str]

Returns

cookie dict

class aioqzone.api.loginman.UPLoginMan(client, uin, pwd)

Returns

exception aioqzone.api.loginman._NextMethodInterrupt

Internal exception represents the condition that the login method is interrupted and the caller could choose the next login method or just to raise a LoginError.
