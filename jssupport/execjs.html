运行 JavaScripts

提供了一个通过调用子进程来执行 JavaScript 的类

警告

在 win32 平台上，导入此模块会导致 asyncio 的事件循环策略被更改为 asyncio.WindowsProactorEventLoopPolicy！

class jssupport.execjs.ExecJS

以下列顺序执行 JavaScript：

  • setup: setup, add_setup()

  • run: run, add_run()

  • post: post, add_post()

If it is required to change executable name, use classvar node. Note that this must be a executable name, so something like bash -c is illegal.

add_post(func, *args, asis=False)

Add a post function to post.

add_run(func, *args, asis=False)

Add a function to run.

add_setup(func, *args, asis=False)

Add a setup function to setup.

bind(expr=None)

Return an partial of .exec, with node set to node, and js set to current javascripts in queue: (setup + run + post).

参数

expr (Union[str, Partial, None]) – _可选的_。如果给定，表达式会被加入 run。

check_all()

运行所有检查，判断当前环境是否可以运行 js。

async static exec(node, js)

使用给定的 node 可执行程序运行 js。

返回类型

str

async get(prop)

Get a property from this environment. run will be cleared.

参数

prop (str) – The property to get.

classmethod loop_policy_check()

On Windows, the default event loop asyncio.ProactorEventLoop supports subprocesses, whereas asyncio.SelectorEventLoop does not.

node: str = 'node'

node executable name. Default as node.

post: List[Union[str, Partial]]

Expressions executed after run. Will not be cleared after executing.

run: List[Union[str, Partial]]

Expressions to be executed, after setup but before post. Will be cleared after executing.

setup: List[Union[str, Partial]]

Expressions executed before run. Will not be cleared after executing.

class jssupport.execjs.Partial(name, *args, asis=False)

A partial like functools.partial. It represents a expression in javascript, which includes a predicate (func) and its arguments.
