伪装一个 Dom 环境

class jssupport.jsdom.JSDOM(*, src='', ua='', location='', referrer='')

JSDOM class provides a complete environment to encounter detection based on collecting runtime environment information.

备注

requires jsdom to be installed.

add_eval(script)

Cache this script into queue. Scripts will be called in order.

返回

None

check_all()

Run all checks to see if the environment is ready to run.

classmethod check_jsdom()

Check if jsdom is installed.

返回

True if jsdom can be imported. Otherwise False.

async eval(script)

Eval a script(expression) throught window.eval().

返回

script(expression) result, if any.

post: List[JsExpr]

Expressions executed after run. Will not be cleared after executing.

run: List[JsExpr]

Expressions to be executed, after setup but before post. Will be cleared after executing.

setup: List[JsExpr]

Expressions executed before run. Will not be cleared after executing.
